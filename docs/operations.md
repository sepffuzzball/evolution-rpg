# Operations

## Backups

Use the Export button in the app to download a JSON backup. Store backups outside the browser before clearing site data.

## Restores

Use the Import button and select a previously exported JSON file.

## Health checks

The Docker image includes an HTTP health check against `/` on port 8080.

## Data retention

Data retention is controlled by browser site storage. Incognito/private sessions and browser cleanup tools may delete character data.
