# Classes
Only humanoids and some half-monsters have classes. These can range from things like "Thug" or "Pickpocket" to "Page", "Knight", "Barbarian", or "Mage". All classes have a rarity associated with them which denotes their general power level.

# Leveling Up
When someone levels up in a class, they receive a bonus to their statistics based on the class's stat-growth ratio, the actual tier record the class is assigned to, and rarity. The tier used here is not the class template's minimum available tier.

Class level-up points: `Tier Class Multiplier * Rarity Multiplier`.

Rarity multiplier:
* Common: 1
* Uncommon: 1.5
* Rare: 2
* Epic: 2.5
* Legendary: 3
* Mythical: 3.5
* Divine: 4
