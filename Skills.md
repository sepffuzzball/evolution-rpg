# Skills
Skills are an important part of *The System* as it's the primary way in which someone uses their MP. Not that all skills require MP, most Passive Skills are weaker but have a constant effect. Typically the higher the MP usage of a skill the more powerful it is.

# Skill vs Spell vs Technique
Skill, spell, and technique are all the same thing - if they are granted by *The System* they are all considered 'Skills' though some archaic ways of calling things would say a that a 'Technique' is not something granted by the system but just how someone does something.

# Skill Sources
Skills can be granted by:
* Racial Tier-ups for Racial Skills
* Class Tier-ups for Class Skills
* Job Tier-ups for Job Skills
* Items (Must be "Set" to receive the Skill associated with an Item)

# Casting Time
All Active skills have a casting time (though some might be "Instantaneous"). This is mostly used for "Spells" cast by classic "Magic User" classes.

# Skill Cooldown
Many Active skills have a Skill Cooldown (which could be seconds, minutes, hours or even days)

# MP Cost
Many Active skills have a Mana Cost in order to use the Skill

# Skill Level
Many skills (Active or Passive) may be able to level up, though not all of them are able to

# Skill Rarity
Skills have a rarity which can and will increase or decrease over time, leveling up, and tiering up.
